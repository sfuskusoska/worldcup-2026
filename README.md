# worldcup-2026

ワールドカップ2026を仲間内で楽しむためのダッシュボード。

## 機能

- 🇯🇵 日本戦の予想投票・リアルタイム応援
- 🌎 全試合表・会場マップ（Phase 2）
- ⏰ 次のキックオフカウントダウン

## 技術スタック

- HTML + Vanilla JS + CSS（フレームワークなし）
- Firebase Realtime Database（Sparkプラン・無料）
- football-data.org API（Freeプラン）
- GitHub Pages ホスティング

## ローカル起動

```powershell
cd D:\ドキュメント\worldcup-2026
python -m http.server 8000
```

ブラウザで `http://localhost:8000` を開く。

## ライセンス

身内向けのため特に設定なし。
